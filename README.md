This is a simple unfinished project for drawing stuff on the terminal (literally). Have a stylus or a mouse have at it. Currently supports simple pen (Braille pixels) or a thick pen (Block pixels).
This project used FTXUI library for easy and clean access to the terminal.
Do take a look at that and enjoy :)

If you wanna run it just git clone this project, run cmake and then make. 
And voila you will have a quick on the spot whiteboard or {$your_terminal_colour} board.
